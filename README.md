## How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/maryqu3en/stack-queue-in-LL.git
   ```

2. Navigate to the project directory:

   ```bash
   cd stack-queue-in-LL
   ```

3. Compile the C files:

   ```bash
   gcc -o main *.c
   ```

4. Run the program:

   ```bash
   ./main
   ```
